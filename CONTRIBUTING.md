# Contributing

Thanks for improving Bottle Rating Generator.

## Development

```bash
npm install
npm run dev
```

## Checks

Run these before opening a pull request:

```bash
npm run lint
npm run format:check
npm run typecheck
npm run test:run
npm run build
```

## Commits

Use Conventional Commits such as `feat: add template picker` or
`fix: handle invalid share data`.

## Issues and PRs

Please include the browser, device, reproduction steps, and screenshots when
reporting mobile interaction issues.
